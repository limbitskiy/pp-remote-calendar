## Remote appointment maker

Can add/remove/edit appointments on a calendar and see some statictical info aboout them.

### Stack:

Server:

```
- MongoDb
```

Client:

```
- Nuxt
- Pinia
```

### How to run:

- clone repo
- run `docker-compose -f calendar-app.yml up -d`
- go to `localhost:3000`
